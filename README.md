# Projeto Codex

Este é um projeto de teste para conectar o ChatGPT Codex ao GitHub.

## Objetivo

Permitir que o ambiente do Codex interaja com este repositório para testes de integração.

## Tecnologias

- Python 3.11
- Requisições HTTP
- GitHub API

## Estrutura

- `/src`: Código-fonte
- `/tests`: Testes automatizados
- `README.md`: Este arquivo

## Autorização

Este repositório está liberado para uso de testes com a aplicação Codex.
